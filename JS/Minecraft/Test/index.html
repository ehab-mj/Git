2D Minecraft Game

This is a simplified 2D version of Minecraft. You can select tools to remove blocks from the world and add them to your inventory. Then, select items from your inventory to place them back in the world. Use the axe for trees, pickaxe for rocks, and shovel for dirt. Grass can be removed with any tool.

How to play:

  • Click a tool to select it.
  • Click a tile in the world to remove it (if the tool matches).
  • Click an item in the inventory to select it.
  • Click an empty tile in the world to place it.
  • Use the reset button to restore the world.
